# Code of Conduct

## Our Commitment

We want this project to be a welcoming, respectful, and collaborative space for everyone.

Contributors, maintainers, and participants are expected to interact in a constructive and professional manner regardless of:
- experience level
- background
- identity
- nationality
- skill level
- communication style

---

# Expected Behavior

Examples of positive behavior include:

- Being respectful and constructive
- Giving helpful feedback
- Being open to different ideas and perspectives
- Focusing on improving the project
- Helping beginners learn and contribute
- Assuming good intent during discussions

---

# Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment or discrimination
- Personal attacks or insulting language
- Trolling or intentionally disruptive behavior
- Public or private intimidation
- Repeated hostile interactions
- Sharing private information without permission
- Spam or unrelated promotional content

---

# Contribution Discussions

Technical disagreements are normal in software development.

Please:
- criticize ideas, not people
- provide reasoning for decisions
- stay constructive during reviews
- avoid unnecessary hostility

---

# Maintainer Responsibilities

Project maintainers are responsible for:
- enforcing this Code of Conduct
- maintaining a respectful environment
- handling reports fairly and professionally

Maintainers may remove, edit, or reject contributions or comments that violate this Code of Conduct.

---

# Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by opening a private issue or contacting the maintainers directly.

All reports will be reviewed and handled appropriately.

---

# Scope

This Code of Conduct applies to:
- GitHub discussions
- issues
- pull requests
- community interactions related to this project

---

# Final Notes

This project values:
- learning
- collaboration
- respectful discussion
- clean and maintainable engineering

Thank you for helping make the community welcoming for everyone ❤️
